🔐 Keshav Cyber Portfolio

A modern cybersecurity portfolio built using Vite, TypeScript, and Tailwind CSS. This project showcases cybersecurity skills, projects, and knowledge in a clean and responsive web interface.

🚀 Features
⚡ Fast performance with Vite
🎨 Styled using Tailwind CSS
🧠 TypeScript for better code quality
📱 Fully responsive design
🔐 Cybersecurity-themed UI
📁 Project Structure
Keshav-Cyber-Portfolio/
│── public/              # Static assets
│── src/                 # Source code
│── index.html           # Entry HTML file
│── package.json         # Dependencies & scripts
│── tailwind.config.js   # Tailwind configuration
│── vite.config.ts       # Vite configuration
🛠️ Installation & Setup
1. Clone the Repository
git clone https://github.com/YOUR_USERNAME/Keshav-Cyber-Portfolio.git
2. Navigate into the Project
cd Keshav-Cyber-Portfolio
3. Install Dependencies
npm install
4. Run the Development Server
npm run dev

👉 Your app will run on something like:

http://localhost:5173/
